# Deployment Notes

This repository is configured as a GitHub Pages user site for:

```text
https://mkurdi21.github.io
```

The Jekyll configuration uses:

```yaml
url: "https://mkurdi21.github.io"
baseurl: ""
repository: "MKurdi21/MKurdi21.github.io"
```

Because the repository is already named `MKurdi21.github.io`, no project-site `baseurl` is configured.
