This project focuses on payment-channel integration testing across ATM, POS, mobile POS, online payment gateway, and card workflows.

The work included validating device behavior, transaction flows, authorization paths, reversals, receipts and data masking, pilot rollouts, go-live readiness, defect retesting, and post-deployment validation. It shaped my interest in security-sensitive systems where defects can affect customers, merchants, and financial operations.
