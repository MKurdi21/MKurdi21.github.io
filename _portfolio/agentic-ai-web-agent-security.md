This project reflects my current Ph.D. research direction under Professor Junjie Zhang. It focuses on web and browser-based AI agents that interact with pages, tools, files, code, credentials, and external services.

Key themes include prompt injection, unsafe tool use, external-content manipulation, credential exposure, multi-step workflow risk, and evaluation methods for autonomous or semi-autonomous agents. The goal is to understand how agentic systems create security risks that are different from standalone language-model use.
