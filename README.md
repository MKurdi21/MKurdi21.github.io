# Mohammad Alkurdi Academic Website

This repository contains the source code for Mohammad Alkurdi's academic portfolio website, built with the AcademicPages Jekyll template.

The website highlights Mohammad's Ph.D. work in Computer Science and Engineering at Wright State University, current research under Professor Junjie Zhang, previous M.S. and early Ph.D. research under Professor Fathi Amsaad, research interests in AI security and agentic AI security, graduate cybersecurity background, selected publications, and professional experience in banking/payment systems QA/UAT.

## Local development

Install dependencies and run:

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open <http://localhost:4000>.

## Notes

This site is based on the AcademicPages template.
